# The Adventure of the Vection

本アプリケーションは、MediaPipeを用いたリアルタイム顔検出により、PCカメラとユーザーとの物理的な距離を測定し、その距離に応じて描画空間を疑似3D移動させるベクション（視覚誘導性自己運動知覚）実験アプリケーションである。レトロゲームエンジンPyxelを使用し、点群による廊下と画像による廊下の2つのモードで視覚効果を検証できる。

---

## 機能概要

* **リアルタイム距離測定**: Webカメラの映像から両眼の間隔（ピクセル結像サイズ）を検出し、ユーザーと画面の実際の距離を推定する。
* **疑似3Dベクション効果**: ユーザーが画面に近づく、または遠ざかる動きに連動して、Pyxel上のカメラ座標（Z軸）が滑らかに移動する。
* **2つの実験モード**: スペースキーによって、点群（ドット）で構成された廊下と、画像で構成された廊下の描画を切り替える。
* **線形補間（LERP）移動**: 顔検出のサンプリング間隔によるガタつきを抑え、カメラ移動を滑らかにする補間処理を実装している。

## 使用方法

### 1. 依存ライブラリのインストール

必要なパッケージをpipでインストールする。

```bash
pip install -r requirements.txt

```

### 2. アプリケーションの起動

メインプログラムを実行する。実行時、PCのWebカメラへのアクセスが求められる。

```bash
python main.py

```

### 3. 操作方法

* **画面との距離変更**: ユーザーがPCカメラに対して前後に動くと、画面内の景色が連動して拡大・縮小（前後移動）する。
* **SPACE キー**: 廊下の描画モードを切り替える（点群モード $\leftrightarrow$ 画像モード）。

## アルゴリズムについて

### 距離推定の原理

人間の平均的な両眼間隔（$63\,\text{mm}$）とカメラの仮想焦点距離を基準とし、検出された両眼のピクセル結像幅（$\text{pixel\_distance}$）から逆算してカメラと顔の物理的距離を測定している。

$$distance\_mm = \frac{\text{REAL\_EYE\_DISTANCE} \times \text{FOCAL\_LENGTH\_PC\_CAMERA}}{\text{pixel\_distance}}$$

計算された距離（$\text{cm}$）をベースに、以下の式でPyxel空間内のカメラターゲットZ座標を決定している。

$$\text{camera\_target\_z} = 15 - \text{distance\_cm} \times 0.5$$
